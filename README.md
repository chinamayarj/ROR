# RubyOnRails

Employee Management

We can add, delete and update employee records.

Developed on RoR 

Ruby Version (2.5)
Rails Version (5.2.3)

Steps to run application.
1) Download the application.
2) bundler install
3) rake db:migrate
4) start server => rails server

Server will be running at port 3000
http://localhost:3000
